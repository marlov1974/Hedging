# Baseloads

Baseloads are standard configurations centered on base exposure.

## Typical component structure

```text
base.sys
base.epad
```

## Description

`base.sys` and `base.epad` are separate market components that together form the base representation.

A Baseloads configuration may use MW as call-off representation and MWh as settlement representation.

## Settlement principle

The contract part calendar converts MW to MWh.
